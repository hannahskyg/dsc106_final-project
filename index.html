Write up:

What have you done so far?

At the moment, we have created and published our GitHub page with our first interactive visualization. We have integrated our first interactive visualization by using d3 to import two datasets from 2019 and 2022 and create a chart that compares the mean U.S. Household net worth between the two years. To make it interactive and more detailed for getting started with the scrollytelling, we implemented a filter checkbox that changes the bar chart to visualize based on the sex of the household head. We also have some simple styling/CSS down to get started with making the website more appealing to the audience.
